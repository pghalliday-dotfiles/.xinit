# xinit

Clone to `~/.xinit`

```
git clone git@github.com:pghalliday-dotfiles/xinit.git ~/.xinit
```

and create `~/.xinitrc`

```
echo "source ~/.xinit/config.sh" > ~/.xinitrc
```

## depends

```
xorg-server
xorg-server-utils
xorg-apps
xorg-init

trayer
xmonad
```
